# Archiel-bot
A "Reply to Hi message" Discord (self-)bot made with Discord.js(-self)

## 🚀 Deploy using "normal" method
1. Clone this repository (you probably already know how to do it)
2. Install all dependencies (`npm install`)
3. Open `.env` and configure every environment variable value
4. Run `node index.js`
